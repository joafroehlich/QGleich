2x² - 7x + 3 = 0      
 
Satz von Vieta: 
 
q = x1 * x2  

- p = x1 + x2 
  
   2x² - 7x + 3 = 0 |:2 
 
x² - 3,5x + 1,5 = 0 
 
q = 1,5 lässt sich zerlegen in 3 * 0,5 ; (-3) * (-0,5) ; 1 * 1,5 ; (-1) oder   
 
(-1,5) * (-1) 

-p = -(-3,5) = 3,5  
 
Da 3 + 0,5 = 3,5 und 3 * 0,5 = 1,5 sind die Lösungen 
 
x1 = 0,5 

x2 = 3
