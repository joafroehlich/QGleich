2x2 - 7x + 3 = 0      
 
Satz von Vieta: 
 
q = x1 * x2  

- p = x1 + x2 
  
   2x2 - 7x + 3 = 0 |:2 
 
x2 - 3,5x + 1,5 = 0 
 
q = 1,5 lässt sich zerlegen in 

3*0,5 ; (-3)*(-0,5) ; 1*1,5 ; oder   
 
(-1,5)*(-1) 

-p = -(-3,5) = 3,5  
 
Da 3 + 0,5 = 3,5 und 3 * 0,5 = 1,5 

sind die Lösungen 
 
x1 = 0,5 

x2 = 3
