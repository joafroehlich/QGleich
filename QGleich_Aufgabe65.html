Quadratische Gleichungen Aufgabe 65

x² + 12x + 27 = 0      
 
Satz von Vieta: 
 
q = x1 * x2  

- p = x1 + x2 
  
x² + 12x + 27 = 0 
 
q = 27 lässt sich ganzzahlig zerlegen in 3 * 9 ; (-3) * (-9) ; 27 * 1 ;  
 
(-27) * (-1) 

-p = -12  
 
Da (-3) + (-9) = -12 und (-9) * (-3) = 27, sind die Lösungen 
 
x1 = -9 

x2 = -3
