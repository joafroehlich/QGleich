Quadratische Gleichungen Aufgabe 93

Ein Rechteck ist doppelt so groß wie ein Quadrat. 
Die eine Seite des Rechtecks ist so groß wie eine Quadratseite, die andere 
ist 12 cm lang. 
Wie groß ist eine Quadratseite?   
 
Die Quadratseite sei x 
 
Das Rechteck ist größer 
 
Fläche des Rechtecks = 12 * x 
 
Fläche des Quadrates x² 
 
    12 * x = 2 * x² |- 12x 
 
 2x² - 12x = 0 
 
 2x(x - 6) = 0 
 
Produkt von Linearfaktoren 
 
    2x = 0  :2 
 
 x1 = 0 keine Lösung, die Quadratseite hätte keine Länge 
 
     x - 6 = 0 |+6 
 
        x2 = 6 cm
