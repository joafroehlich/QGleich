Quadratische Gleichungen Aufgabe 98

Eine Küche ist dreimal so lang wie breit. 
Sie kann mit 108 ganzen quadratischen Platten ausgelegt werden. 
Die Platten haben eine Kantenlänge von 50 cm. 
Wie lang und wie breit ist die Küche?  

Eine Platte hat eine Fläche von 0,5 m * 0,5m = 0,25 m² 

Die 108 Platten bedecken eine Fläche von 
108 * 0,25 m² = 27 m² =  Fläche der Küche. 

Die Breite sei x m, dann ist die Länge 3 * x m. 

x * 3x = 27 

3x² = 27 | :3 

x² = 9 |√ 

x1 = 3 m = Breite der Küche 

x2 = - 3 keine Lösung, negative Länge 

Die Länge beträgt 3 * 3 m  = 9 m
