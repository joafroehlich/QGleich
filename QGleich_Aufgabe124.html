Ein Bus kostet für eine Fahrt 120 €. 
Die Kosten werden auf die Teilnehmer gleichmäßig 
verteilt. Hätte sich die Teilnehmerzahl um 2 
erhöht, wären die Kosten pro Teilnehmer um 
0,25 Cent gesunken. 
Wie viel Personen haben an der Fahrt teilgenommen?

Die Teilnehmerzahl sei x

                       120

                        x 

2 Teilnehmer mehr x + 2 

                              120
Kosten pro Teilnehmer jetzt ------- €
                             x + 2

Ersparnis 0,25 € pro Teilnehmer

 120      120

  x      x + 2     

Hauptnenner x(x + 2)

 120      120

  x      x + 2     

120(x + 2) - 120x = 0,25x(x + 2)

120x + 240 - 120x = 0,25x² + 0,5x |-240

0,25x² + 0,5x - 240 = 0 | :0,25

x² + 2x - 960 = 0

p, q - Formel

p = 2 ; q = -960
